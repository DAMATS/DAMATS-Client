What's new in the GeoPortal

Here we provide a very short overview about the changes. Nevertheless, we strongly recommend to read the respective sections in the Online Help to find out more…

TimeSlider:

The new TimeSlider provides a Zoom In/Out functionality using the mouse wheel. Inside the lower area (Date area) the TimeSlider can be dragged to change the time shown. Symbols indicate product availability at a point in time. Full symbol → Product is within viewing area; Empty symbol → Product is outside current viewing area. A Click on a symbol will zoom you to the product's extent. You can select a Time-Period by clicking and dragging the mouse in the upper part of the TimeSlider(→ red bar) e.g. used for Download. This time selection can be enlarged / shortened ('⇔'-icon) or shifted ('hand'-icon) by dragging. Please read more details in the Online Help…

Snow / Ice & Glaciers:

Contains all Snow and River/Lake Ice products / Contains all Glacier images, Glacier Lakes and Glacier Outlines products. A DatasetSeries can be activated by clicking the rectangle left of the Name. The DoubleArrow allows resorting the layers by simply dragging them up or down. Please read more details in the Online Help…

Tools:

Area-of-Interest (AOI) selection using Polygons or Rectangles. Import / Export / Save your AOI, as a geoJSON file. Please read more details in the Online Help…

Download:

Before a download is possible you have to select at least one DatasetSeries, a AOI, and a Time-Period. The download of a Time-Series of products is still provided. New output formats are offered. New output projections are offered. Please read more details in the Online Help…

Server:

OGC support has changed to OGC EO-WMS and EO-WCS (WCS 2.0) requests only! Due to the large number of products (∼20.000) plain WMS-GetCapbilities requests exceeded any reasonable size (>16MB) and creation time (~20min). Valis EO-WMS requests need to provide a DatasetSeries name plus a Time-Stamp in the GetMap Request. CryoLand now also supports the OGC WCS Interface Standard – CRS Extension, version 1.0.0. This also required a change in the GetCoverege request syntax. Please read more details in the Online Help, where also a link to new request examples, showing the new syntax, is provided.
